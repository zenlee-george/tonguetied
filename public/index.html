Path Method Description
Users Table
/users/ GET Gets all of the users
/users/:id GET Gets a single by id
/users/ POST Adds a user
/users/:id DELETE Deletes a by id
/users/:id PATCH Edits all information of a user by id
Products Table
/product/ GET Gets all of the products
/product/:id GET Gets a single product by id
/product/ POST Adds a product
/product/:id DELETE Deletes a product by id
/product/:id PATCH Edits all information of a product by id
