WELCOME

Home

HTTP Methods Path Description
GET / returns to the home page

Users

HTTP Methods Path Description
GET /users returns all the users
GET /users/:id returns a user based on their id
POST /users/signUp registers a user on the database
POST /users/signIn logs in a user
PATCH /users/:id updates user details based on their id
DELETE /users/:id deletes user from the database

Products

HTTP Methods Path Description
GET /products returns all the products
GET /products/:id returns product based on their id
POST /products/add adds a product to the database
PATCH /products/:id updates product details based on their id
DELETE /products/:id deletes a product from the database

Carts

HTTP Methods Path Description
GET /users/:userID/cart returns all the carts
GET /users/:userID/cart/:cartID returns cart based on their id
POST /users/:userID/cart adds an item to cart
PATCH /users/:userID/cart/:cartID updates an item in cart
DELETE /users/:userID/cart/:cartID deletes an item in cart
DELETE /users/:userID/cart/:cartID deletes all items in the cart
