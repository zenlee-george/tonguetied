Users

HTPP Methods Path Description
GET /users returns all the users
GET /users/:id returns a user based on their id
POST /users/signUp it registers a user on the database
POST /users/signIn for login in a user
PATCH /users/:id Updates a user details based on their id
DELETE /users/:id deletes user on the database

Products

HTPP Methods Path Description
GET /products returns all the products
GET /products/:id returns product based on their id
POST /products/add it add a product on the database
PATCH /products/:id Updates a product details based on their id
DELETE /products/:id deletes a product on the database
